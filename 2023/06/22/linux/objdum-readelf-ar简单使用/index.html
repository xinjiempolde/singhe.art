objdum-readelf-ar简单使用

发表于 2023-06-22 更新于 2024-12-11 分类于 linux 阅读次数：

file

file能够查看文件的类型

size

能够查看ELF文件的代码段、数据段和BSS段的长度(dec表示3个段长度的和的十进制，hex表示长度和的十六进制)

objcopy

使用objcopy工具，可以将图片、MP3音乐等作为目标文件中的一个段

objdump

objdump -h

objdump -h能够输出ELF文件中各个段的信息

objdump -s

objdump -s能够将所有段的内容以十六进制的方式打印出来

objdump -d

objdump -d可以将所有包含指令的段反汇编

objdump -r

查看重定位段的信息

objdump -t

查看静态库或者目标文件中的符号相关信息

readelf

readelf -h

readelf -h能够查看ELF头的信息

readelf -S

能够查看ELF文件的段，和objdump -h作用相同

readelf -s

能查看ELF文件中的符号段的信息

nm也能达到类似的效果

readelf -l

查看可执行文件的Segment header

readelf -d

查看".dynamic"段的内容：

readelf -sD

查看动态符号表的内容和哈希表的内容

c++filt

能够解析C++修饰过的符号名称

ar

ar -t

查看静态库中包含哪些目标文件

ldd

ldd可以查看程序主模块或者一个共享库依赖于哪些共享库
